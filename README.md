# AB---Artificial-Brain
# ANCORA IN FASE DI SVILUPPO PRIVATA | DEMO FUNZIOINANTE PUBBLICATA

- completa mappatura delle aree di brodmann

- utilizzo di pacchetti JSON per la comunicazione tra ogni area del AB. L'interpretazione del pacchetto viene lasciata all'IA ricevente.

- Il AB è dotato di pensiero autonomo si dall'inizio, e di un sistema di premiazione (sentimenti).

- mappatura cerebrale su `AB.pdf`.

# FEATURES IN SVILUPPO:
- modello ML creato tramite phyton che gestisca ogni area separatamente.
  
- funzionamento da chat bot (in questo momento il modello generico (llama3) risponde solo a stimoli autonomi e attività di fondo).
