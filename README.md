# AB---Artificial-Brain
# ANCORA IN FASE DI SVILUPPO PRIVATA | AGGIORNEÒ LA REPOSITORY UNA VOLTA CREATA UNA DEMO FUNZIOINANTE E COMPLETA
- utilizzo di un modello ML semplice creato tramite phyton, uguale per ogni parte del AB. Testando, dopo abbastanza crescita, il modello si adatta alle regole interne della sua area specifica.
  
- utilizzo di pacchetti JSON per la comunicazione tra ogni area del AB. L'interpretazione del pacchetto viene lasciata all'IA ricevente (crea risultati più genuini ed organici).
  
- il programma funziona come chat bot, ma essendo che tutte le aree partono da zero (incluso l'area di Broca, l'area di Wernicke, e l'area di Geschwind), il AB imparerà non solo la lingua, ma anche a scrivere in chat,
  tramite gli input dell'utente
  
- Il AB è dotato di pensiero autonomo si dall'inizio, ed è dotato di funzioni neuroplastiche (come il sonno), e di un sistema di premiazione (sentimenti), tridimensionale, permettendo una interpretazione più complessa di
  un sistema scalare (come passare da una scala di grigi, ad uno spettro di colori)
  
- tutto viene gestito dal NeuroBus, completamente scritto in Java. Si occupa dello scambio, e nell'instradaggio dei pacchetti JSON.
